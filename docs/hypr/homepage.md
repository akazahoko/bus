# Themes
Tools
- GTK2/3/4: `nwg-look`
- Qt5: `qt5ct-kde` (AUR)
- Qt6: `qt6ct-kde` (AUR)

## [Catppuccin](https://catppuccin.com/) (GTK/Qt)
(Available from AUR)
- GTK: `catpuuccin-gtk-theme-{flavor}`
- Qt: `catpuuccin-plasma-colorscheme-{flavor}`

Avaliable flavors:
- `latte` (White)
- `frappe` (Dark)
- `macchiato` (Darker)
- `mocha` (Darkest)

*Extra: [Catppuccin Color Palette](https://catppuccin.com/palette/)*
